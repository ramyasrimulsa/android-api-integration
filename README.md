# android-api-integration 🚀

A simple and resilient Android application built with **Kotlin** that demonstrates modern REST API integration using **Retrofit2** and **MVVM Architecture**.

---

## 📁 Project Structure

This repository isolates only the 8 core files and modified configuration scripts required to build the network framework:

```text
📁 app/
├── 📄 build.gradle.kts             # Dependencies (Retrofit, Gson, Lifecycle)
└── 📁 src/main/
    ├── 📄 AndroidManifest.xml      # Internet permission flag
    ├── 📁 res/layout/
    │   └── 📄 activity_main.xml    # UI screen components
    └── 📁 java/com/example/backendintegrationapp/
        ├── 📁 data/
        │   ├── 📄 Post.kt          # JSON Data Object Model
        │   ├── 📄 ApiService.kt    # HTTP Request Endpoints (GET/POST)
        │   └── 📄 RetrofitClient.kt# Main Network Connection Instance
        └── 📁 ui/
            ├── 📄 MainActivity.kt  # View and click event binder
            └── 📄 MainViewModel.kt # Coroutine data handler & error catcher
```

---

## 🛠️ Tech Stack & Features

* **Kotlin Coroutines**: Executes network calls smoothly in the background without freezing the UI.
* **LiveData**: Automatically updates the user interface as soon as new data arrives from the server.
* **Crash-Free Error Handling**: Intercepts `IOException` states natively. If the user loses internet connectivity, the application displays a friendly warning message instead of crashing.

---

## 🚦 How to Test the App

1. **Fetch Data (GET)**: Requests live mock data from the cloud server and populates the text block.
2. **Send Data (POST)**: Simulates uploading a new post and prints out the unique ID generated by the server.
3. **Offline Mode**: Turn off your phone's Wi-Fi/Data and tap any button. The application cleanly catches the drop and prompts a `NETWORK ERROR: Check your connection.` message.
